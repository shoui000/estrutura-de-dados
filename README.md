# Estrutura de Dados

Esse repositório armazenará todos os códigos de aula de laboratório da disciplina de Estrutura de Dados no curso de Ciência da Computação na UFES. Além disso, conterá exercícios feitos durante a leitura e estudo baseado no livro "Entendendo Algoritmos" de Aditya Y. Bhargava. O livro não tem relação direta com a disciplina a não ser pelo conteúdo.
